Why do I need front-end testing?

There are untold number of subtle errors that can occur on the front-end.

  • Minor CSS changes that throw things off
  • Changes to JS files that break things
  • Aggregates changing when not necessary
  • Performance regressions

Tenets of Front-end Testing

  • JavaScript Unit Testing
  • Integration Testing
  • Performance/Stress Testing
  • CSS & JS Linting
  • Automated Browser Testing
  • Visual Regression Testing
  • Accessibility Testing
  • User Testing

What can I do about it?

  • Create a performance budget
  • Write automated UI tests
  • Learn Visual Regression Testing
  • Set up JavaScript linting
